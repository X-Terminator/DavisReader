Project for reading serial data from Davis Vantage Pro 2 Weather Station (via Serial Data Logger) and pushing the data to MQTT for use by Home Assistant.
Tested on Wemos D1 Mini board (ESP8266).

#### Used Libraries
- Arduino IDE 1.8.8 (https://www.arduino.cc/en/main/software)
- Arduino core for the ESP8266 2.4.2 (https://github.com/esp8266/Arduino)
- ArduinoJson 5.13.4 (https://github.com/bblanchon/ArduinoJson.git)
- NTPClient 3.1.0 (https://github.com/arduino-libraries/NTPClient)
- PubSubClient 2.7 (http://pubsubclient.knolleary.net)
